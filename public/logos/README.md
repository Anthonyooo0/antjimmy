## Logo Specifications

- **Format**: PNG (with transparent background preferred)
- **Size**: Recommended 400-600px wide
- **Color**: Logos will be displayed in grayscale by default and show color on hover

## Note

The logos are displayed with grayscale filter and reduced opacity by default. On hover, they transition to full color with full opacity.
